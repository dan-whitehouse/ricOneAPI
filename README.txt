How to Use
====================
1. On the GitHub project page, click the HTTPS clone URL button on the right hand side of the project.
2. Open Eclipse
3. Click File in the menu bar and then click import
4. In the pop-up window, double click the folder labeled "Git"
5. From there select "Projects from Git" and then click the "Next" button.
6. Click "Close URI" and then click "Next"
7. Paste the link from step 1 into the field labeled "URL"
8. Follow the next few steps as to where you wish to put the project (We suggest your eclipse workspace)
